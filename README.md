## 📌 Overview

The **Diabetes Health Indicator System** is a web-based healthcare application that predicts diabetes risk using medical and lifestyle indicators.

The system combines:

- 🧠 Machine Learning Prediction
- 📊 Interactive Data Visualization
- 📁 CSV Dataset Analysis
- 📄 PDF Report Generation

The project helps users understand diabetes risk based on healthcare factors like BMI, HbA1c, Blood Glucose, Age, Smoking History, Hypertension, and Heart Disease.
